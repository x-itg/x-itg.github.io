# 硬件引脚配置表

版本: 1.2.0
最后更新: 2026-04-15

## MCU: STM32F407VG

## I2C接口

### I2C1 - 传感器总线
| 引脚 | 功能 | 连接设备 | 说明 |
|------|------|----------|------|
| PB6 | I2C1_SCL | SCL | 上拉4.7KΩ |
| PB7 | I2C1_SDA | SDA | 上拉4.7KΩ |

### I2C2 - 扩展总线
| 引脚 | 功能 | 连接设备 | 说明 |
|------|------|----------|------|
| PB10 | I2C2_SCL | 预留 | - |
| PB11 | I2C2_SDA | 预留 | - |

## UART接口

### USART2 - RS485通信
| 引脚 | 功能 | 说明 |
|------|------|------|
| PA2 | USART2_TX | RS485 DI |
| PA3 | USART2_RX | RS485 RO |
| PA1 | USART2_DE | RS485 DE/RE |

### USART1 - 调试串口
| 引脚 | 功能 | 说明 |
|------|------|------|
| PA9 | USART1_TX | ST-LINK |
| PA10 | USART1_RX | ST-LINK |

## GPIO

### 指示灯
| 引脚 | 功能 | 说明 |
|------|------|------|
| PC13 | LED_GREEN | 运行指示 |
| PC14 | LED_RED | 错误指示 |

### 按键
| 引脚 | 功能 | 说明 |
|------|------|------|
| PA0 | KEY_USER | 用户按键 |

## SPI接口

### SPI1 - OLED显示
| 引脚 | 功能 | 说明 |
|------|------|------|
| PA5 | SPI1_SCK | OLED_SCLK |
| PA6 | SPI1_MISO | - |
| PA7 | SPI1_MOSI | OLED_SDIN |
| PB2 | OLED_CS | 片选 |
| PB0 | OLED_DC | 数据/命令 |
| PB1 | OLED_RES | 复位 |

## ADC通道

| 通道 | 功能 | 说明 |
|------|------|------|
| PA0 | ADC_IN0 | 电池电压监测 |
| PA1 | ADC_IN1 | 预留 |

|--------|------|------|
| TIM2 | 系统滴答 | 1ms |
| TIM3 | PWM输出 | 1kHz |
| TIM4 | 编码器 | - |
| TIM5 | 保留 | - |
